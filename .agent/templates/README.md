# Agent Task Templates

Use these templates to reduce ambiguity and speed up handoff between roles.

## Available Templates

- `tech-lead-task.md`
- `coder-task.md`
- `researcher-task.md`
- `qa-task.md`

## Usage

1. Copy the template for the target role.
2. Fill inputs/scope/constraints.
3. Keep output paths and verification commands explicit.
