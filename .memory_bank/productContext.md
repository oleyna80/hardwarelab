# Product Context

## What is HardwareLab?

HardwareLab is an **Amazon affiliate website** focused on hardware product reviews. The site helps users discover and compare hardware products, earning commission through Amazon Associate program referrals.

## Target Audience

- Tech enthusiasts researching hardware purchases
- Developers looking for SBCs (Single Board Computers)
- Home users seeking NAS, mini PCs, gaming consoles
- International audience (EN, FR, RU, DE speakers)

## Content Categories

| Category | Examples |
|----------|----------|
| Mini PCs | Mac mini M4, Intel NUC |
| SBCs | Orange Pi 5 Plus, Raspberry Pi |
| NAS | Synology DS423+ |
| Consoles | PlayStation 5 Pro |
| Gaming | Gaming PCs, peripherals |
| Monitors | Gaming monitors, productivity displays |
| AI Workstations | High-performance AI builds |

## Monetization Model

1. **Amazon Affiliate Links** — Primary revenue via Amazon Associate program
2. **Regional Tags** — Different affiliate tags per region (US, UK, DE, FR)
3. **Compliance** — All affiliate content follows Amazon Operating Agreement

## Brand Voice

- **Honest**: Balanced reviews with pros/cons
- **Technical**: Detailed specifications and benchmarks
- **Accessible**: Complex topics explained clearly
- **Multilingual**: Same quality across all 4 languages
