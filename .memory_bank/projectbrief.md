# Project Brief

Фундаментальный документ, определяющий цели и рамки проекта HardwareLab.

---

## Цель проекта

Создать **высокопроизводительный affiliate-сайт** для обзоров железа с монетизацией через Amazon Associates.

**Домен:** [hardwarelab.org](https://hardwarelab.org)

---

## Scope (Рамки)

### Включено
- Обзоры hardware продуктов (Mini PC, SBC, NAS, консоли, мониторы)
- 4 языка: English, French, Russian, German
- Affiliate-монетизация (Amazon Associates)
- SEO-оптимизация для органического трафика

### Исключено
- E-commerce (прямые продажи)
- User-generated content
- Форум или комьюнити

---

## Ключевые ограничения

| Ограничение | Значение |
|-------------|----------|
| VPS ресурсы | 1 vCPU, 512MB RAM |
| Affiliate compliance | Amazon Operating Agreement |
| Performance target | Lighthouse > 90 |
| Content quality | Экспертные обзоры, не AI-spam |
